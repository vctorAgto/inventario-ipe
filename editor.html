Editor do inventário

Acesso restrito a quem cuida do inventário. As alterações aqui gravam um commit direto no repositório GitHub — a tela pública reflete essas mudanças assim que o GitHub Pages publicar (geralmente ~1 minuto).

Entrar no editor

Peça o nome e a senha de acesso a quem administra o inventário.

Configuração avançada (só para quem administra o repositório)
Deixe em branco pra usar os valores padrão deste repositório. A "senha de acesso" é, por baixo dos panos, um token do GitHub (fine-grained, permissão só de "Contents: Read and write" neste repositório) — quem administra gera esse token em github.com → Settings → Developer settings → Fine-grained tokens, e compartilha com os editores como se fosse uma senha única. Qualquer pessoa com essa senha consegue editar o repositório: não compartilhe fora do grupo de editores.
