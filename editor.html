Editor do inventário

As alterações aqui gravam um commit direto no repositório GitHub. A tela pública (index.html) reflete essas mudanças assim que o GitHub Pages publicar (geralmente ~1 minuto).

Acesso do editor (preencher uma vez por navegador)
A senha fica salva apenas neste navegador (localStorage) — nunca é gravada no repositório. Peça a senha de acesso a quem administra o inventário. Seu nome aparece só no histórico de alterações do GitHub, pra facilitar saber quem atualizou o quê.
Configuração avançada (só para quem administra o repositório)
Deixe em branco pra usar os valores padrão deste repositório. A "senha de acesso" acima é, por baixo dos panos, um token do GitHub (fine-grained, permissão só de "Contents: Read and write" neste repositório) — quem administra gera esse token em github.com → Settings → Developer settings → Fine-grained tokens, e compartilha com os editores como se fosse uma senha única. Qualquer pessoa com essa senha consegue editar o repositório: não compartilhe fora do grupo de editores.
