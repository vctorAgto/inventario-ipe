Editor do inventário

As alterações aqui gravam um commit direto no repositório GitHub. A tela pública (index.html) reflete essas mudanças assim que o GitHub Pages publicar (geralmente ~1 minuto).

Configuração do repositório (preencher uma vez por navegador)
O token fica salvo apenas neste navegador (localStorage) — nunca é gravado no repositório. Crie um token em github.com → Settings → Developer settings → Fine-grained tokens, com acesso restrito só a este repositório e permissão "Contents: Read and write". Qualquer pessoa com este token consegue editar o repositório: não compartilhe o link desta tela publicamente, apenas com quem for atualizar o inventário.
